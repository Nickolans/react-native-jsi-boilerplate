# react-native-jsi-boilerplate

Boilerplate for module using jsi.

## Installation

```sh
npm install react-native-jsi-boilerplate
```

## Usage

```js
import { multiply } from "react-native-jsi-boilerplate";

// ...

const result = await multiply(3, 7);
```
